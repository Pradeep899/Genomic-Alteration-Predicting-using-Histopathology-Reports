# Genomic-Alteration-Predicting-using-Histopathology-Reports
NLP pipeline predicting FDA-drug-targetable cancer mutations from pathology report text — no genomic sequencing needed. Combines TF-IDF and Bio_ClinicalBERT embeddings in an ensemble model across 8,796 TCGA patients (74% AUC, up to 92% per-gene), with a clinical decision report mapping predictions to approved targeted therapies.
